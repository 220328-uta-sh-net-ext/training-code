Functions in JS

Methods vs functions Methods and functions depicts the behavior of the program. But how are they different from each other??

There are many types of functions in JS.

  • Named Functions
  • Anonymous Functions
  • Arrow Functions
  • Prototype Functions
  • IIFE (Immediately Invoked Function expression) Functions
  • Callback Functions

Methods Functions
Methods belong to behaviour of an object, which means it will reside within a class Functions depicts the behavior of a program and doesn't neccessarily have to be in the class, it can reside just inside the program
